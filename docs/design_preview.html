The 4 Icon Variants — When to Use Each

Variant A
Bare Glyph

A small monochrome symbol (◆ ◇ ▸) or icon at 12–14px, no container. Uses the section's accent color at full opacity.

Use for: Section headers, page titles, nav labels
Variant B
Inline Icon

Emoji or icon at 16–18px placed inline with text, no container. Let the icon's natural color carry the weight.

Use for: Learn topic rows, challenge list items, profile rows
Variant C
Status Pill

Horizontal pill with a 5px dot + label. Tinted background, colored text, no box. Communicates state (New, Active, Done).

Use for: Category tags, progress status, challenge type labels
Variant D
Solid Circle

Solid filled circle at 28–32px. Used sparingly — one per section max. Signals the primary recommended action.

Use for: Primary CTA items only (Start, Resume, Featured)

5-item max for mobile nav. Leaderboard is a social/status feature — accessible from Profile tab or Dashboard card. Translate is a core BIM feature used actively during sessions, so it earns the nav slot. New order: Home · Learn · Translate · Challenges · Profile

Display font swapped from Familjen Grotesk → Bricolage Grotesque 800. Scale corrected from 1.14× to 1.33× steps. Screen titles now 26–28px creating clear dominance over card titles at 12–14px. Body copy in Nunito 400 — warmer, more accessible than Lexend.
02
Dashboard — Level as Hero
Gradient welcome card removed. Level number rendered at 76px as the visual anchor — users identity with their level; let it dominate. XP progress inlined next to it. Stats become a 3-col mini grid. "Continue" uses horizontal scroll so multiple lessons are visible without full-card height.
03
Learn — Magazine Layout
Categories as horizontal scroll with mini progress bars — scannable at a glance. Featured "Resume" lesson gets a dedicated large card with its own progress bar and CTA. Remaining lessons in compact text-first list. No icon-box monoculture at any level.
04
Universal — Zero AI Tells
No gradient cards. No tinted icon boxes. No equal-height card grids. No bounce animations. Settings use a grouped plain list (iOS/Android native convention). Profile uses a stat grid created with 1px gaps — no cards within cards. Quiz result uses a 88px score number as the emotional anchor.
