# gesturaclaude

A new Flutter project.

## About

This project is a sign language mobile application that have bidirectional translation with a complete learning modules for signs language
- Collaboration/Permission for assets in this application are directly given by Malaysian Federation of the Deaf (MFD) from their Bahasa Isyarat Malaysia (BIM) Sign Bank
- Millions of thanks for the MFD for the permission to use their BIM sign bank to complete this project
